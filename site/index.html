to make it all much more secure and easier

We're fully focused on privacy and security, so we deliver several programs that ensure this in your daily use.

Another branch of these efforts is to deliver the most complete and fully functional desktop, with some familiar programs and others that may you don't know yet but they are very easy to use.

All of these features are backed by Debian bookworm as the base system.

Features

The Debian that you already know, but with superpowers.

CLI easy and fast addons

With addons like atuin, liquidprompt, fd, bat and many others, your bash will certainly be much easier and confortable to deal with.

Program package format

Do you like flatpaks, snaps or appimages? You're fully covered, we provide very easy ways to install all of them! (take a look at am) Distrobox is already included.

Multimedia content creation

If you are a content creation, this is the right distro for you, the apt and am repositories have tons of packages to easily install the latest versions of REAPER, KDEnLive, GIMP, Blender and many others.

Linux gaming

You're looking for fun? With the special version of Steam, you just need to install Your games, chill and play! It's the same compatibility and performance from Steam Deck.
